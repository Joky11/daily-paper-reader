## 摘要
随着大型语言模型成为内容生成的标准后端，实际的溯源需求日益要求多比特水印。在供应商内部部署中，一个关键要求是消息对称性：消息本身不应系统性地影响文本质量或验证结果。词汇划分水印在低熵解码中可能破坏消息对称性：某些消息被分配到大部分概率质量，而其他消息则被迫使用尾部的词元。这使得嵌入质量和消息解码准确率依赖于具体消息。我们提出了 QuantileMark，一种在连续累积概率区间 $[0, 1)$ 内嵌入消息的白盒多比特水印。在每个步骤中，QuantileMark 将该区间划分为 $M$ 个等概率区间，并严格从分配给目标符号的区间内采样，从而确保无论上下文熵如何，都具有固定的 $1/M$ 概率预算。对于检测，验证者在教师强制条件下重建相同的划分，计算潜在区间上的后验概率，并聚合证据进行验证。我们证明了消息无偏性——这一性质保证了在对消息取平均后可恢复基底分布。这为生成端的对称性提供了理论基础，而等概率设计进一步促进了在检测端消息之间的证据强度均匀性。基于 C4 continuation 和 LFQA 的实证结果显示，QuantileMark 在多比特恢复和检测鲁棒性方面均超越强基线，同时对生成质量的影响可忽略不计。我们的代码可在 GitHub 获取（https://github.com/zzzjunlin/QuantileMark）。

## Abstract
As large language models become standard backends for content generation, practical provenance increasingly requires multi-bit watermarking. In provider-internal deployments, a key requirement is message symmetry: the message itself should not systematically affect either text quality or verification outcomes. Vocabulary-partition watermarks can break message symmetry in low-entropy decoding: some messages are assigned most of the probability mass, while others are forced to use tail tokens. This makes embedding quality and message decoding accuracy message-dependent. We propose QuantileMark, a white-box multi-bit watermark that embeds messages within the continuous cumulative probability interval $[0, 1)$. At each step, QuantileMark partitions this interval into $M$ equal-mass bins and samples strictly from the bin assigned to the target symbol, ensuring a fixed $1/M$ probability budget regardless of context entropy. For detection, the verifier reconstructs the same partition under teacher forcing, computes posteriors over latent bins, and aggregates evidence for verification. We prove message-unbiasedness, a property ensuring that the base distribution is recovered when averaging over messages. This provides a theoretical foundation for generation-side symmetry, while the equal-mass design additionally promotes uniform evidence strength across messages on the detection side. Empirical results on C4 continuation and LFQA show improved multi-bit recovery and detection robustness over strong baselines, with negligible impact on generation quality. Our code is available at GitHub (https://github.com/zzzjunlin/QuantileMark).

## 一、研究问题与背景  

- **研究动机**：  
  - 随着大型语言模型（LLM）广泛用于文本生成、问答与代码创作，内容溯源（provenance）成为必要能力，用于追踪生成来源、防范滥用与确保合规。  
  - 现有水印技术主要解决**零比特（是否为AI生成）检测问题**，而实际应用需要**多比特水印**以嵌入用户ID、模型版本、时间戳等元数据。  
- **核心问题**：  
  - 在多比特水印中，不同消息应当得到均等的生成质量与检测可靠度，即具有**消息对称性（message symmetry）**。  
  - 传统的“词汇划分（vocabulary partition）”水印方案在低熵条件下导致严重的不对称：高频词被集中分配给部分消息，其他消息被迫生成低概率词元，从而降低文本质量并削弱检测强度。

## 二、方法论：QuantileMark 的设计思想与技术细节  

### 1. 核心思想  

- QuantileMark 摒弃了词汇空间划分思路，改为在**累积概率分布区间 [0,1)** 上操作。  
- 通过**等概率（equal-mass）分箱量化（quantile partitioning）**，每个消息符号都占有固定的 1/M 概率预算，从而在理论上保证：  
  - 生成时的信息嵌入不会导致概率分布失衡；  
  - 检测时每个消息获得等量证据。  

### 2. 算法流程（描述性）  

- **(1) 消息编码与位置分配**：  
  将消息 $S=(s_1,\dots,s_H)$ 拆分为多个符号，每个符号 $s_i\in\{0,\dots,M-1\}$。  
  每个生成步骤 $t$ 通过密钥 $K$ 和局部上下文 $g_t$ 派生对应的符号索引 $i_t$ 与一个随机置换 $\varphi_t$。  

- **(2) 嵌入阶段（Embedding）**：  
  - 将累计概率区间 [0,1) 按等量概率划分为 M 个子区间 $B_r=[r/M,(r+1)/M)$。  
  - 目标符号通过置换映射到目标分箱 $r_t^\star=\varphi_t(s_{i_t})$。  
  - 从该子区间内部均匀采样 $u_t\sim U(B_{r_t^\star})$，并选取覆盖该随机值的词元作为输出。  
  - 此策略保证每个符号的采样概率严格为 1/M，从而实现消息无偏（message-unbiasedness）。  

- **(3) 检测阶段（Detection）**：  
  - 检测者在白盒条件下利用教师强制（teacher forcing）重建模型的原始分布 $p_0(\cdot|h_t)$。  
  - 对于观测到的词元 $x_t$，依据其在分箱区间中的重叠质量 $\mu_t(x_t,r)$ 计算后验 $p_t(r)=\mu_t(x_t,r)/p_0(x_t|h_t)$。  
  - 将得到的后验转化为**对数后验比 (Log Posterior Odds, LPO)**，并通过累计求和来解码完整消息 $\hat S$。  
  - 若平均 LPO 超过阈值 $\theta$，即可判定存在水印信号。  

- **(4) 理论性质**：  
  - 通过期望计算证明“消息无偏性（message-unbiasedness）”与“密钥无偏性（cipher-unbiasedness）”。  
  - 保持平均生成分布不变，即即使嵌入水印，整体模型输出统计量与原模型一致。

## 三、实验设计与评测设定  

### 1. 数据集与任务  

- **C4 Continuation（新闻续写）**：基于 Llama-2-7B。  
- **LFQA（ELI5 长文问答）**：基于 Llama-3.1-8B-Instruct。  

### 2. 对比方法  

- **MPAC**（Yoo et al., 2024）：词汇划分多比特水印。  
- **StealthInk**（Jiang et al., 2025）：分布保持、黑盒可检水印。  
- **No watermark**：无水印基准。  

### 3. 实验指标  

- Bit Accuracy（比特恢复率）  
- AUC、TPR@1%FPR（检测能力）  
- PPL（困惑度，生成质量）  
- 检测时间（Efficiency）  

### 4. 实验设置  

- 消息长度：24比特（分12个符号）。  
- 生成长度：300个词元。  
- 解码参数：top-k=128，温度 τ=1.0。  
- 每个任务：500 个水印样本 + 500 个非水印样本。

## 六、主要结论与发现  

- QuantileMark 在两个任务上均显著优于对照组：  
  - **C4：** 比特恢复率 0.9893，AUC≈0.9995；  
  - **LFQA：** AUC≈0.9997，TPR@1%FPR=1.0；  
  - 困惑度接近无水印文本（ΔPPL<0.3）。  
- 对水印去除攻击具有较强抵抗力，尤其在复制/同义词替换下仍可高精度恢复。  
- 对解码参数变化（温度、top-k）表现出平滑鲁棒性。  
- 与下游任务（翻译、摘要、问答质量）结果几乎无差别，验证其生成质量中立性。

- **结构对称设计：**
