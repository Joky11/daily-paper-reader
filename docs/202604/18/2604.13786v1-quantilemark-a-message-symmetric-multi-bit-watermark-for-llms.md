## 摘要
随着大型语言模型成为内容生成的标准后端，对内容溯源的实际需求日益要求实现多比特水印。在模型提供方内部部署的情形下，一个关键要求是消息对称性：消息本身不应系统性地影响文本质量或验证结果。基于词汇划分的水印方案在低熵解码时可能破坏消息对称性：某些消息被分配到大部分概率质量，而其他消息则被迫使用尾部词，从而导致嵌入质量和消息解码准确率依赖于消息本身。我们提出 QuantileMark，这是一种白盒多比特水印方法，将消息嵌入到连续累积分布区间 $[0,1)$ 中。在每一步，QuantileMark 将该区间划分为 $M$ 个等概率子区间，并严格从分配给目标符号的子区间中采样，从而确保无论上下文熵如何，每个符号都有固定的 $1/M$ 概率预算。在检测阶段，验证方通过教师强制（teacher forcing）重建相同划分，计算潜在区间的后验概率，并汇聚证据以进行验证。我们证明了消息无偏性（message-unbiasedness），这一性质保证了在对所有消息取平均时能恢复原始分布。这为生成端的对称性提供了理论基础，而等概率设计进一步在检测端促进了跨消息证据强度的均匀性。在 C4 continuation 和 LFQA 任务上的实验结果表明，该方法在多比特恢复和检测鲁棒性方面优于强基线方法，对生成质量的影响可以忽略。我们的代码已在 GitHub 发布（https://github.com/zzzjunlin/QuantileMark）。

## Abstract
As large language models become standard backends for content generation, practical provenance increasingly requires multi-bit watermarking. In provider-internal deployments, a key requirement is message symmetry: the message itself should not systematically affect either text quality or verification outcomes. Vocabulary-partition watermarks can break message symmetry in low-entropy decoding: some messages are assigned most of the probability mass, while others are forced to use tail tokens. This makes embedding quality and message decoding accuracy message-dependent. We propose QuantileMark, a white-box multi-bit watermark that embeds messages within the continuous cumulative probability interval $[0, 1)$. At each step, QuantileMark partitions this interval into $M$ equal-mass bins and samples strictly from the bin assigned to the target symbol, ensuring a fixed $1/M$ probability budget regardless of context entropy. For detection, the verifier reconstructs the same partition under teacher forcing, computes posteriors over latent bins, and aggregates evidence for verification. We prove message-unbiasedness, a property ensuring that the base distribution is recovered when averaging over messages. This provides a theoretical foundation for generation-side symmetry, while the equal-mass design additionally promotes uniform evidence strength across messages on the detection side. Empirical results on C4 continuation and LFQA show improved multi-bit recovery and detection robustness over strong baselines, with negligible impact on generation quality. Our code is available at GitHub (https://github.com/zzzjunlin/QuantileMark).

## 一、研究动机与背景

- **核心问题**：随着大型语言模型（LLMs）广泛用于内容生成、问答、代码编写等任务，文本出处追踪与内容归属成为迫切需求。尤其在实际平台的“提供方内部”部署中，要求不仅检测出生成来源，还需支持多比特水印来编码诸如用户 ID、模型版本、时间戳等元数据。  
- **痛点**：传统基于词汇划分的水印方法（Vocabulary Partition Watermark）在低熵场景下会导致 *消息对称性* 被破坏。部分消息可能对应高概率词，嵌入容易且不影响质量，而其他消息被迫使用尾部词，使得文本质量和检测准确率随消息内容变化。  
- **目标**：在多比特水印场景下实现 **消息无偏性（Message Symmetry）**，即不同消息在生成质量和验证效果上应有一致且公平的表现。  
- **研究意义**：提供一种既保持生成自然度、又具有稳定检测性能的 *白盒多比特水印方案*，服务于内容溯源、合规检查和内部安全追踪。

## 二、方法论概述

### 核心思想

- 提出 **QuantileMark**：一种在 **累积分布函数 (CDF)** 空间操作的 *白盒多比特水印*。
- 将传统在词表空间的操作转移到连续概率区间 $[0,1)$，实现等概率嵌入，保证每个符号（代表消息位）占用固定概率预算，不再受文本熵或分布尖锐度影响。

### 关键技术流程

#### 1. 嵌入阶段（Embedding）

- 将累积分布区间划分为 $M=2^m$ 个等概率子区间（bins）。
- 每个消息符号 $s_i$ 映射到目标子区间 $B_r$，并只从该区间中采样。
- 对应的 *水印分布*：  
  $$p_{wm}(v|h_t, r) = M \cdot \mu_t(v, r)$$  
  其中 $\mu_t(v,r)$ 表示词 $v$ 的概率区间与分区 $r$ 的重叠面积。
- 该设计确保每个符号拥有相同的概率预算（1/M），实现消息对称嵌入。

#### 2. 检测阶段（Detection）

- 基于 **教师强制（Teacher Forcing）** 重建相同的累积分布。
- 计算每个词对应各分区的后验概率：  
  $$p_t(r) = \frac{\mu_t(x_t, r)}{p_0(x_t|h_t)}$$
- 将分区后验转化为 **对数后验比 (LPO)** 作为证据：  
  $$LPO_t(r) = \log \frac{p_t(r)}{1-p_t(r)}$$
- 通过密钥派生的序列位置分配 $i_t$ 聚合证据，逐位解码消息：  
  $$\hat{s}_h = \arg\max_s \sum_{t: i_t=h} LPO_t(\phi_t(s))$$
- 最终，整体检测得分为平均 LPO，超过阈值即判定为水印文本。

#### 3. 理论性质

- **消息无偏性 (Message-Unbiasedness)** 证明：在所有可能消息平均下，模型的生成分布与原始分布一致。
- **密码无偏性 (Cipher-Unbiasedness)**：在随机化密钥但固定消息情况下，模型分布也保持不变。
- 两者确保水印在统计层面“看不出差异”。

## 六、主要结论与发现

- QuantileMark 实现了理论上严格的消息无偏性与对称性。
- 在两大任务上：
  - **Bit Accuracy ≥ 0.95**，**AUC ≈ 0.999**，检测性能接近上限。
  - 文本困惑度（PPL）几乎不高于无水印版本，生成质量保持。
- 对抗鲁棒性：
  - 抗同义替换与复制拼接效果显著；但对于删除与改写攻击敏感。
- 性能随符号分辨率 m 呈合理退化，m=2–3 为最优折中。
- 与基线相比，QuantileMark 在恢复准确率、高鲁棒性与均一检测性能上均优胜。

## 七、方法与实验亮点

- **结构创新**：从离散词汇空间迁移到连续概率空间的“等质量分区”，从根本上解决了低熵导致的消息不平衡问题。
- **理论完备**：明确定义并证明了两种无偏性（message-unbiasedness 与 cipher-unbiasedness），提供数学保障。
- **检测强度**：采用 LPO（Log Posterior Odds）作为对称证据累积，优于传统似然比 LLR。
- **高效性**：检测阶段可与生成阶段同速执行，成本低，适合内部大规模部署。
- **公平性与安全性**：消息独立，符号分布均匀，不会因消息不同而影响文本质量或识别概率。

- **鲁棒性限制**：在强 paraphrase 或大量编辑情况下，水印同步性受破坏，检测性能显著下降。
