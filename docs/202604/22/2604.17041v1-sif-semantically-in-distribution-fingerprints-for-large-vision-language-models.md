## 摘要
大型视觉语言模型（LVLM）的公开可获取性引发了关于未经授权的模型再利用和知识产权侵权的严重担忧。现有的所有权验证方法通常依赖于语义异常的查询或分布外响应作为指纹，这些指纹容易被对手检测和移除。我们通过语义发散攻击（Semantic Divergence Attack, SDA）揭示了这一漏洞。该攻击通过测量可疑模型与参考模型之间的语义发散来识别和过滤指纹查询，表明现有指纹不具备语义保持性，因此易于被检测和绕过。为克服这些局限，我们提出了 SIF（Semantically In-Distribution Fingerprints），一种无需修改模型参数的非侵入式所有权验证框架。SIF 引入语义对齐指纹蒸馏（Semantic-Aligned Fingerprint Distillation, SAFD），该方法将文本水印信号转移到视觉模态中，从而生成语义一致且带有指纹的响应。此外，鲁棒指纹优化（Robust-Fingerprint Optimization, RFO）通过模拟最坏情况下的表示扰动来增强鲁棒性，使指纹在模型微调和量化等改动下仍然保持有效。在 LLaVA-1.5 和 Qwen2.5-VL 上的大量实验表明，SIF 具有很强的隐蔽性和鲁棒性，为 LVLM 的版权保护提供了一种可行的解决方案。代码可在 https://github.com/UCF-ML-Research/SIF-VLM-Fingerprint 获取。

## Abstract
The public accessibility of large vision-language models (LVLMs) raises serious concerns about unauthorized model reuse and intellectual property infringement. Existing ownership verification methods often rely on semantically abnormal queries or out-of-distribution responses as fingerprints, which can be easily detected and removed by adversaries. We expose this vulnerability through a Semantic Divergence Attack (SDA), which identifies and filters fingerprint queries by measuring semantic divergence between a suspect model and a reference model, showing that existing fingerprints are not semantic-preserving and are therefore easy to detect and bypass. To address these limitations, we propose SIF (Semantically In-Distribution Fingerprints), a non-intrusive ownership verification framework that requires no parameter modification. SIF introduces Semantic-Aligned Fingerprint Distillation (SAFD), which transfers text watermarking signals into the visual modality to produce semantically coherent yet fingerprinted responses. In addition, Robust-Fingerprint Optimization (RFO) enhances robustness by simulating worst-case representation perturbations, making the fingerprints resilient to model modifications such as fine-tuning and quantization. Extensive experiments on LLaVA-1.5 and Qwen2.5-VL demonstrate that SIF achieves strong stealthiness and robustness, providing a practical solution for LVLM copyright protection. Code is available at https://github.com/UCF-ML-Research/SIF-VLM-Fingerprint

## 二、方法论：SIF 框架

### 1. 整体思想
SIF（Semantically In-Distribution Fingerprints）是一种**非侵入式指纹验证框架**，确保指纹嵌入在模型的自然语义空间内，不影响其正常输出。  
核心包括：
- **语义对齐指纹蒸馏（SAFD）**
- **鲁棒指纹优化（RFO）**

### 2. 语义对齐指纹蒸馏（Semantic-Aligned Fingerprint Distillation, SAFD）

- **目的**：将文字水印信号（原用于文本所有权保护）转移到视觉输入，使模型在自然语义下生成带指纹的响应。  
- **关键机制**：
  1. 设原始图像为 \(x\)，触发图像为 \(x' = x + \delta\)，其中 \(\|\delta\|_\infty ≤ \varepsilon\)，保证视觉外观基本不变。
  2. 在生成文本时，模型输出分布 \(p_\theta(y_t | x', p, y_{<t})\) 被优化以：
     - 保持与教师模型输出的语义一致。
     - 在输出的 token 中隐含文本水印列表 \(G_t\) 的统计偏置。
  3. 使用两个损失函数联合优化：
     - **水印对齐损失** \(L_{wm}\)：鼓励生成 token 落在水印词表内；
     - **语义一致性损失** \(L_{ce}\)：限制输出接近教师响应，防止语义漂移。
  4. 总损失公式：  
     \[
     L(x') = \lambda_{wm} L_{wm}(x') + \lambda_{ce} L_{ce}(x')
     \]
- **结果**：生成的触发图像视觉上几乎无差异，但在正常解码下输出带隐蔽水印信号，可用于版权验证。

### 3. 鲁棒指纹优化（Robust-Fingerprint Optimization, RFO）

- **目标**：提升指纹在模型微调、量化下的稳定性。  
- **核心思想**：在 SAFD 优化中模拟最坏情况的表示扰动，以增强指纹对模型内部变化的鲁棒性。  
- **步骤**：
  1. 计算梯度 \(g_\ell = ∇_{h_\ell}L_{base}(x')\)，标识最易破坏指纹的方向。
  2. 生成梯度对齐扰动 \(ϵ^*_\ell\)，注入中间激活 \(h_\ell(x')\)。
  3. 在扰动环境下重新优化，使指纹能抵御后续模型改动。
- **效果**：模拟量化、微调造成的表征漂移，使指纹信号更坚固、不易失效。

## 六、主要结论与发现

- **发现 1**：现有指纹方法均易被语义发散攻击（SDA）检测和替换，对版权保护作用削弱。
- **发现 2**：SIF 指纹在**语义分布内**运行，与正常查询无异常（PPL≈自然分布），攻击难以识别。
