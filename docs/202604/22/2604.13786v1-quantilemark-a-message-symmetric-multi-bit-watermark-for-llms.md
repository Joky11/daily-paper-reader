## 摘要
随着大型语言模型逐渐成为内容生成的标准后端，实际的来源追溯越来越需要多比特水印。在供应商内部部署中，一个关键要求是消息对称性：消息本身不应系统性地影响文本质量或验证结果。词汇划分水印在低熵解码中可能破坏消息对称性：某些消息被分配到大部分概率质量，而其他消息则被迫使用尾部词元。这使得嵌入质量和消息解码准确性依赖于消息内容。我们提出了 QuantileMark，这是一种白盒多比特水印，它将消息嵌入到连续的累积概率区间 $[0,1)$ 中。在每一步中，QuantileMark 将该区间划分为 $M$ 个等质量的箱，并严格从分配给目标符号的箱中采样，从而在上下文熵无论如何变化时都保持固定的 $1/M$ 概率预算。对于检测，验证者在教师强制模式下重建相同划分，计算潜在箱的后验概率，并聚合证据以进行验证。我们证明了消息无偏性，这一性质确保在对消息取平均时可以恢复基础分布。这为生成端对称性提供了理论基础，而等质量设计进一步促进了检测端在不同消息间证据强度的均匀性。我们在 C4 continuation 和 LFQA 上的实验证明了相较强基线，QuantileMark 提升了多比特恢复能力和检测鲁棒性，同时对生成质量影响可忽略。我们的代码已在 GitHub（https://github.com/zzzjunlin/QuantileMark）公开。

## Abstract
As large language models become standard backends for content generation, practical provenance increasingly requires multi-bit watermarking. In provider-internal deployments, a key requirement is message symmetry: the message itself should not systematically affect either text quality or verification outcomes. Vocabulary-partition watermarks can break message symmetry in low-entropy decoding: some messages are assigned most of the probability mass, while others are forced to use tail tokens. This makes embedding quality and message decoding accuracy message-dependent. We propose QuantileMark, a white-box multi-bit watermark that embeds messages within the continuous cumulative probability interval $[0, 1)$. At each step, QuantileMark partitions this interval into $M$ equal-mass bins and samples strictly from the bin assigned to the target symbol, ensuring a fixed $1/M$ probability budget regardless of context entropy. For detection, the verifier reconstructs the same partition under teacher forcing, computes posteriors over latent bins, and aggregates evidence for verification. We prove message-unbiasedness, a property ensuring that the base distribution is recovered when averaging over messages. This provides a theoretical foundation for generation-side symmetry, while the equal-mass design additionally promotes uniform evidence strength across messages on the detection side. Empirical results on C4 continuation and LFQA show improved multi-bit recovery and detection robustness over strong baselines, with negligible impact on generation quality. Our code is available at GitHub (https://github.com/zzzjunlin/QuantileMark).

- **总体目标：**  
  构造一种水印算法，使得嵌入不同消息的过程及检测性能**完全对称、无偏**，在任何上下文熵下均保持一致的嵌入质量与可验证性。

## 二、方法论：核心思想与关键技术

### 1. 方法概览
- 提出**QuantileMark**：一种白盒多比特水印框架，基于**累积分布函数(CDF)的等质量划分**实现。
- 主要设计目标：  
  - 保证每个消息符号具有相同的采样概率预算。  
  - 避免词汇划分带来的概率偏差。  
  - 实现**生成端的消息无偏性（message-unbiasedness）**与**检测端的均匀证据强度**。

### 2. 核心机制

- **输入与设置：**  
  对长度为 \(T\) 的生成序列，每步嵌入符号 \(s_i \in \{0, \ldots, M-1\}\)，其中 \(M=2^m\)，即每符号对应 \(m\) 比特信息，共 \(H\) 个符号。

- **等质量量化划分（Equal-Mass Quantile Partitioning）：**  
  将累积分布区间 \([0,1)\) 平均分为 \(M\) 个区间（bin）：  
  \[
  B_r = [r/M, (r+1)/M)
  \]
  - 每个区间均对应一个符号。  
  - 嵌入时仅从目标区间内采样随机数 \(u \sim U(B_{r^\*})\)，输出覆盖该点的词元。  
  - 保证每个符号在任意上下文中均占总概率的 \(1/M\)。

- **公式定义：**
  - **重叠质量（overlap mass）：**  
    \(\mu_t(v,r)=|I_t(v)\cap B_r|\)，度量词元概率区间与量化区间的交叠长度。
  - **水印分布：**  
    \(p_{wm}(v|h_t,r)=M \cdot \mu_t(v,r)\)。  
    表示若目标为第 \(r\) 号区间时的输出分布。

- **检测阶段（白盒）：**
  - 在教师强制模式下重建 \(p_0(\cdot|h_t)\)，计算观测词元落入各区间的后验：  
    \[
    p_t(r) = \frac{\mu_t(x_t,r)}{p_0(x_t|h_t)}
    \]
  - 将每步的证据转化为**对数后验比（LPO）**：
    \[
    LPO_t(r)=\log \frac{p_t(r)}{1-p_t(r)}
    \]
  - 按嵌入位置聚合并最大化求得消息：
    \[
    \hat{s}_h = \arg\max_s \sum_{t:i_t=h} LPO_t(\phi_t(s))
    \]
  - 生成序列级检测分数 \(T(x)=\frac{1}{T}\sum_t LPO_t(\hat{r}_t)\)，与阈值比较以判断水印存在。

- **理论性质：**  
  - 证明**消息无偏性**：对所有词元 \(v\)，平均所有消息后有  
    \[
    E_s [p_{wm}(v|h,R)] = p_0(v|h)
    \]
  - 即平均后分布回收基础模型，确保不引入系统性偏差。

## 六、主要结论与发现

- **量化信道实现消息对称性**：每消息享有相同概率预算，消除了低熵分布下的系统性偏差。  
- **检测性能显著提升**：  
  - C4 与 LFQA 上 Bit Accuracy 分别达 0.9893 与 0.9500，AUC≈1.0，TPR@1%FPR≈1。  
  - 超过 MPAC 与 StealthInk 且几乎无质量损失（PPL≈无水印基线）。  
- **鲁棒性良好**：对局部编辑（同义/删除/拼接）保持高 AUC（>0.94），但对强改写仍会失效。  
- **消息扩展性**：检测性能在32–56位消息下保持稳健，性能下降平滑。  
- **白盒检测高效稳定**：无论采样温度或top‑k误差均平滑退化，无灾难性失效。  
- **生成质量保持**：下游任务翻译/摘要指标几乎与无水印持平。
