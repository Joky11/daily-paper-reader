## 摘要
大型视觉语言模型（LVLMs）的公开可访问性引发了对模型未经授权复用和知识产权侵权的严重担忧。现有的所有权验证方法通常依赖语义异常的查询或分布外响应作为指纹，这些指纹容易被攻击者检测并移除。我们通过语义分歧攻击（Semantic Divergence Attack, SDA）揭示了这一漏洞，SDA通过衡量嫌疑模型与参考模型之间的语义分歧来识别和过滤指纹查询，显示出现有指纹无法保持语义一致，从而易于检测和规避。为了解决这些局限性，我们提出了SIF（Semantically In-Distribution Fingerprints，语义上分布内指纹），一种无需修改模型参数的非侵入式所有权验证框架。SIF引入语义对齐指纹蒸馏（Semantic-Aligned Fingerprint Distillation, SAFD），将文本水印信号转移到视觉模态中，以生成语义一致但带有指纹的响应。此外，鲁棒指纹优化（Robust-Fingerprint Optimization, RFO）通过模拟最坏情况下的表示扰动来增强鲁棒性，使指纹在诸如微调和量化等模型修改下依然保持韧性。在LLaVA-1.5和Qwen2.5-VL上的大量实验表明，SIF在隐蔽性和鲁棒性方面均表现出色，为LVLM的版权保护提供了切实可行的解决方案。代码可在 https://github.com/UCF-ML-Research/SIF-VLM-Fingerprint 获取。

## Abstract
The public accessibility of large vision-language models (LVLMs) raises serious concerns about unauthorized model reuse and intellectual property infringement. Existing ownership verification methods often rely on semantically abnormal queries or out-of-distribution responses as fingerprints, which can be easily detected and removed by adversaries. We expose this vulnerability through a Semantic Divergence Attack (SDA), which identifies and filters fingerprint queries by measuring semantic divergence between a suspect model and a reference model, showing that existing fingerprints are not semantic-preserving and are therefore easy to detect and bypass. To address these limitations, we propose SIF (Semantically In-Distribution Fingerprints), a non-intrusive ownership verification framework that requires no parameter modification. SIF introduces Semantic-Aligned Fingerprint Distillation (SAFD), which transfers text watermarking signals into the visual modality to produce semantically coherent yet fingerprinted responses. In addition, Robust-Fingerprint Optimization (RFO) enhances robustness by simulating worst-case representation perturbations, making the fingerprints resilient to model modifications such as fine-tuning and quantization. Extensive experiments on LLaVA-1.5 and Qwen2.5-VL demonstrate that SIF achieves strong stealthiness and robustness, providing a practical solution for LVLM copyright protection. Code is available at https://github.com/UCF-ML-Research/SIF-VLM-Fingerprint

## 二、方法论：SIF 框架总体设计

### 1. 核心思想
提出 **SIF（Semantically In-Distribution Fingerprints）** 框架，通过“语义内分布的隐形指纹”实现对 LVLM 的非侵入式版权验证。  
其核心理念是：  
> 将文本生成领域的水印思想迁移到图像输入层面，使模型在面对自然输入时生成“语义正常但含隐性指纹”的响应。

### 2. 方法总体流程
SIF 包含两个关键技术模块（见 Figure 3 示意）：

#### (1) 语义对齐指纹蒸馏（Semantic-Aligned Fingerprint Distillation, SAFD）
- **目标**：将原用于文本生成的水印信号蒸馏进视觉输入（图像）中。  
- **做法**：  
  1. 选取一张普通图像 \( x \) 和用户指令 \( p \)。  
  2. 生成带微扰的触发图像 \( x' = x + \delta \), 约束 \(\|\delta\|_{\infty} \le \epsilon\)，保持图像视觉自然。  
  3. 优化触发图像，使得模型生成的响应既：
     - 对齐教师模型的语义输出；
     - 又更倾向于选取水印 token 列表 \( G_t \)。  
  4. 优化目标：
     \[
     L(x') = \lambda_{wm} L_{wm}(x') + \lambda_{ce} L_{ce}(x'),
     \]
     其中 \(L_{wm}\) 为水印对齐损失，\(L_{ce}\) 为语义一致性损失。  
  5. 在优化中，仅调整输入图像像素，不改动模型参数。

#### (2) 鲁棒指纹优化（Robust-Fingerprint Optimization, RFO）
- **目标**：增强指纹在模型经过量化或微调后的稳定性。  
- **思想**：模拟最坏情况的潜在表征扰动。  
- **过程**：
  1. 对中间层激活 \(h_\ell(x')\) 计算梯度 \(g_\ell = \nabla_{h_\ell} L_{base}(x')\)。  
  2. 构造与梯度方向一致、范数受限的扰动 \( \epsilon^\star_\ell = \rho g_\ell / \sqrt{\sum_j ||g_j||_2^2} \)。  
  3. 注入扰动后继续更新输入 \(x'\)，形成对抗式稳健优化。  
- **结果**：触发图像能抵御表示漂移，保持水印指纹不丢失。

#### (3) 验证阶段
- 对嫌疑模型输入触发图像+提示；
- 若其输出文本带有可检测的水印信号（通过 z-score 统计检验），则判定该模型为“源自原模型”。

## 三、实验设计

### 1. 模型与任务
- **目标模型**：LLaVA-1.5-7B 与 Qwen2.5-VL-7B；
- **攻击设置**：
  - **量化**（4-bit / 8-bit）；
  - **全量微调**（含 HuggingFace 发布的多个变体，如 LlavaMix、TikZ、GUI-Actor、ARC-AGI-1）；
  - **自建微调**：V7W、TextVQA、Paintingform、MathV；
- **检测指标**：Fingerprint Matching Rate (FMR)，即水印检测成功率。

### 2. 数据与基准
- **触发数据集**：选自 ImageNet 2012 验证集 + AMBER 文本生成基准；
- **对比方法**：
  - IF（Instruction Fingerprint）
  - Proflingo
  - PLA
  - RNA, DIM, CroPA, Ordinary Attack
- **评价维度**：  
  - 鲁棒性（量化与微调后 FMR）  
  - 隐蔽性（SDA 攻击检测率、查询困惑度 PPL）  
  - 可靠性（在无关模型上的误触发率）  

### 3. 攻击验证
- 提出了 **Semantic Divergence Attack (SDA)** 用以衡量现有方法的脆弱性；
- 结合 GPT-4.1 作为语义判别器评估交互“是否可疑”。

## 六、主要发现与结论

1. **现有方案脆弱性**：Proflingo、PLA、IF 等依赖语义异常，可被 SDA 迅速识别并过滤。  
2. **SIF 成效**：
   - **高鲁棒性**：在 4-bit 量化后 FMR 仍达 0.49（LLaVA）和 0.88（Qwen）；在完整微调模型中仍能有效识别。  
   - **高隐蔽性**：触发查询的 PPL 接近自然分布，被 SDA 标记率 < 5%，远低于对比方法（>85%）。  
   - **高可靠性**：所有非相关模型上 FMR = 0，几乎无误报。  
3. **理论贡献**：首次实现“语义一致 + 分布内”的 LVLM 指纹蒸馏与检测闭环。

## 七、优点与创新点

- **非侵入式**：不需更改模型权重，适合已发布模型。  
- **语义对齐**：指纹嵌入保持语义流畅，不破坏输出质量。  
- **显著稳健性**：能抵抗微调、量化、修剪、输入噪声等多种修改。  
- **检测可解释**：统计水印检测（z-score）具可验证性。  
- **实现开源**：提供完整代码与实验配置，复现性较强。

## 八、不足与局限

- **法律效力有限**：指纹只能作为“所有权初筛”，不足以在法律层面单独确证。  
- **算力成本**：单次指纹生成约 6 分钟，生成大规模触发集仍具时间开销。

- **安全对抗**：若攻击者掌握大规模样本与检测机制，仍可能设计针对性去除攻击。
