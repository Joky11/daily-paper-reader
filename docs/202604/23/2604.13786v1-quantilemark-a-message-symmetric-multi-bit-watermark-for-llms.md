## 摘要
随着大型语言模型成为内容生成的标准后端，实际的来源追溯越来越需要多比特水印。在提供方的内部部署中，一个关键需求是消息对称性：消息本身不应系统性地影响文本质量或验证结果。词汇划分水印在低熵解码时可能破坏消息对称性：某些消息被分配了大部分概率质量，而其他消息则被迫使用低概率的尾部词元。这使得嵌入质量和消息解码准确度依赖于具体消息。我们提出了 QuantileMark，这是一种白盒多比特水印，它在连续累计概率区间 $[0, 1)$ 内嵌入消息。在每一步中，QuantileMark 将该区间划分为 $M$ 个等概率分箱，并严格从分配给目标符号的分箱中进行采样，从而无论上下文熵如何都能确保固定的 $1/M$ 概率预算。对于检测，验证器在教师强制（teacher forcing）条件下重建相同的分区，计算潜在分箱的后验概率，并汇总证据以进行验证。我们证明了消息无偏性（message-unbiasedness），这一性质确保在对消息取平均时能够恢复基础分布。这为生成端的对称性提供了理论基础，同时等概率设计还促进了检测端跨消息的证据强度均匀分布。基于 C4 续写和 LFQA 的实验证明，该方法在多比特恢复和检测稳健性方面优于强基线，同时对生成质量的影响可忽略不计。我们的代码已在 GitHub 发布（https://github.com/zzzjunlin/QuantileMark）。

## Abstract
As large language models become standard backends for content generation, practical provenance increasingly requires multi-bit watermarking. In provider-internal deployments, a key requirement is message symmetry: the message itself should not systematically affect either text quality or verification outcomes. Vocabulary-partition watermarks can break message symmetry in low-entropy decoding: some messages are assigned most of the probability mass, while others are forced to use tail tokens. This makes embedding quality and message decoding accuracy message-dependent. We propose QuantileMark, a white-box multi-bit watermark that embeds messages within the continuous cumulative probability interval $[0, 1)$. At each step, QuantileMark partitions this interval into $M$ equal-mass bins and samples strictly from the bin assigned to the target symbol, ensuring a fixed $1/M$ probability budget regardless of context entropy. For detection, the verifier reconstructs the same partition under teacher forcing, computes posteriors over latent bins, and aggregates evidence for verification. We prove message-unbiasedness, a property ensuring that the base distribution is recovered when averaging over messages. This provides a theoretical foundation for generation-side symmetry, while the equal-mass design additionally promotes uniform evidence strength across messages on the detection side. Empirical results on C4 continuation and LFQA show improved multi-bit recovery and detection robustness over strong baselines, with negligible impact on generation quality. Our code is available at GitHub (https://github.com/zzzjunlin/QuantileMark).

## 一、研究背景与核心问题

- **研究动机**  
  随着大型语言模型（LLMs）成为内容生成的核心后端，其生成文本的**来源可追溯性（provenance）**成为关键需求。这涉及到追踪文本来源、防止虚假信息、支持政策合规和权责管理等。  
  现有方法多为**零比特水印**（仅检测“是否人为生成”），但在真实场景中需要**多比特水印**来嵌入用户 ID、模型版本等内部标识。

- **关键问题：消息对称性（message symmetry）**  
  在多比特水印中，不同消息若引起文本质量或检测结果的系统性差异，将导致：
  - 某些消息生成较自然，另一些则显著劣化；
  - 检测精度依赖具体消息，削弱整体可靠性。  
  现有**词汇划分式水印（vocabulary partition watermark）**在低熵解码时会打破这种对称性，因为概率质量在不同消息间分布极不均匀。

## 二、方法论与核心技术

### 1. 方法总览

- **提出方法：QuantileMark**
  - 属于**白盒多比特水印**（white-box multi-bit watermark）。
  - 通过在**累计概率区间 [0,1)** 上进行**等概率分箱（equal-mass partitioning）** 实现。
  - 每个比特符号对应一个固定质量的分箱，确保无论上下文熵如何，每条消息均获得**相同概率预算（1/M）**。

### 2. 核心思想与算法流程

1. **累积分布映射**  
   将模型当前步的 token 概率 $p_0(\cdot | h_t)$ 映射到区间 [0,1)，形成累积概率区间。  
   每个 token 占据一个长度为其概率质量的子区间。

2. **等质量分箱**  
   将 [0,1) 均分为 $M = 2^m$ 个等宽分箱 $B_r = [r/M, (r+1)/M)$。  
   目标消息符号 $s_{it}$ 映射到对应的分箱 $r_t^* = \phi_t(s_{it})$。

3. **嵌入阶段（Embedding）**
   - 从目标分箱内采样 $u_t \sim U(B_{r_t^*})$；
   - 选择覆盖该采样点的 token 输出；
   - 新的采样分布定义为 $p_\text{wm}(v|h_t, r) = M \, \mu_t(v, r)$，其中 $\mu_t(v, r)$ 为该 token 在分箱中的概率重叠质量。

4. **检测阶段（Detection）**
   - 验证端在 **教师强制（teacher forcing）** 下重建基模型分布；
   - 计算观察 token 所属分箱的**后验概率**
     $$ p_t(r) = \frac{\mu_t(x_t, r)}{p_0(x_t|h_t)} $$
   - 通过 **Log Posterior Odds (LPO)** 聚合跨时间的证据，选择最大累计得分的符号 $\hat{s}_h$ 作为解码结果。

5. **消息无偏性理论（Message-Unbiasedness）**
   - 证明：对任意上下文 $h_t$ 和 token $v$，在对消息取平均时，
     $$ \mathbb{E}_s[p_\text{wm}(v|h_t,\phi_t(s))] = p_0(v|h_t) $$
   - 意味着：在平均意义下，嵌入水印不会扭曲模型分布，保证生成质量不随消息而异。

  白盒检测时间与模型推理时延同量级，适配提供方内部系统。
