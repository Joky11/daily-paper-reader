## 摘要
近期令人兴奋的一系列研究致力于开发大型语言模型（LLM）输出的水印方法。我们专注于可被证明为不可检测的水印方式——即在不改变 LLM 输出分布的情况下，能够在输出中嵌入一个水印，从而识别出该输出是由特定的 LLM 生成的。此外，这种水印对于潜在的攻击者而言应难以移除，即使攻击者可以对带水印的输出执行编辑、插入或删除操作。的确，近期的研究（Christ 等人 [COLT'24]，Christ 等人 [CRYPTO'24]，Golowich 等人 [NeurIPS'24]）展示了如何开发针对常数比例替换甚至常数比例任意编辑具有鲁棒性的此类方案。然而，这些工作对 LLM 输出中的熵水平做出了较强假设。尤其是，它们都要求恒定的熵率——也就是说，在输出的足够长子串中，需有恒定比例的 token 具有至少 O(log |T|) 的经验熵，其中 T 为 token 字母表。此外，Golowich 等人还要求 T 的大小大于安全参数。在本研究中，我们探讨在每个 token 的熵仅为常数的情况下，是否也能对 LLM 输出进行水印，从而去除对字母表大小或安全参数的依赖。在这种条件下，我们构造了：

- 一种在随机替换下具有鲁棒性的水印方案（基于与 Christ 等人 [CRYPTO'24] 相同的次指数级 LPN 假设）；
- 一种在随机替换和随机删除下均具有鲁棒性的水印方案，前提是满足以下任一条件：额外的启发式假设——即 LLM 的输出仅引入随机错误（类似于 Christ 等人 [CRYPTO'24] 所作的假设），或构造出一种对抗替换和随机删除均鲁棒的伪随机纠错码。

## Abstract
A recent and exciting thread of work focuses on developing methods for watermarking the output of large language models (LLMs). We focus on provably undetectable watermarking-that is, schemes that do not alter the output distribution of the LLM, yet enable embedding a watermark in the output that identifies the output as having been generated by the particular LLM. Furthermore, the watermark should be hard to remove by an adversary that may potentially edit, insert, or delete tokens from the watermarked output. Indeed, recent work (Christ et al. [COLT'24], Christ et al. [CRYPTO'24], Golowich et al. [NeuroIPS'24]) shows how to develop such schemes that are robust against a constant fraction of substitutions, or even against a constant fraction of arbitrary edits.   These works, however, make strong assumptions on the entropy present in the output of the LLM. Most notably, they all require constant entropy rate-that is, a constant fraction of the tokens in a sufficiently long substring of the output need to have empirical entropy at least O(log |T|), where T is the alphabet of tokens, and Golowich et al. additionally require T to be larger than the security parameter. In this work, we consider whether we can also watermark the outputs of LLMs when the per-token entropy is just a constant, discarding the dependence on the alphabet size or security parameter. In this regime, we construct:   - A watermarking scheme robust against random substitutions (assuming subexponential LPN, as in Christ et al. [CRYPTO'24])   - A watermarking scheme robust against random substitutions and random deletions, given either the additional heuristic assumption that the output of the LLM only introduces random errors (analogous to the assumption made by Christ et al. [CRYPTO'24]) or a construction of a pseudorandom error-correcting code robust to adversarial substitutions and random deletions.

## 一、研究核心问题与背景

- **研究动机**：  
  随着大型语言模型（LLMs）广泛用于内容生成，其生成文本日趋自然与人类创作难以区分。为了区分“AI生成”与“人类创作”，**水印（watermarking）**成为重要解决方案。  
  现有工作通常要求 LLM 输出具有较高熵（每个 token 的生成分布需足够随机），但在现实语言中，很多场景下 token 熵往往很低（例如固定语句模板、重复词）。高熵假设限制了水印技术的普遍适用性。

- **核心研究问题**：  
  作者探讨：  
  > 当 LLM 的输出仅具有“低熵水平”（例如每个 token 只有常数级别的熵）时，是否仍可实现**不可检测且鲁棒的水印机制**？  
  这直接挑战此前工作中“必须要高熵”的假设。

- **研究意义**：  
  论文从理论上拓展了水印方案的适用范围，使低熵输出（例如常规句法、有限词汇场景）也能被检测与防篡改。

## 二、方法论：核心思想与技术流程

### 1. 总体思路
- 为保持**不可检测性**，水印嵌入不能改变 LLM 输出分布；
- 通过使用 **伪随机纠错码（PRC, Pseudorandom Error-Correcting Code）** 嵌入隐秘比特序列，以实现：
  - **伪随机性**：使水印输出在统计上与原始输出无法区分；
  - **鲁棒性**：保证水印在随机替换 / 删除 / 编辑攻击下仍可识别。

### 2. 技术关键点

- **哈希映射思想**：  
  将 LLM 的大规模 token 字母表 \( T \) 映射到二进制比特空间，通过哈希函数 \( h: T \to \{0,1\} \)。  
  每个 token 被哈希成 0 或 1，从而可与 PRC 码字的对应比特进行关联。

- **嵌入算法（Watermark 函数）核心机制**：
  1. 从 LLM 输出概率分布 \( D \) 中抽样两个独立 token \( t_0, t_1 \)；
  2. 若 \( h(t_0) \neq h(t_1) \)，则选择使得 \( h(t_b) = c_i \) 的 \( t_b \)（其中 \( c_i \) 是待嵌入 PRC 的第 i 位比特）；
  3. 若 \( h(t_0) = h(t_1) \)，则随机选择一个 token 输出；
  4. 保证分布无偏，同时在高熵 token 上有较大概率嵌入成功。

- **检测算法（Detect 函数）**：
  - 使用相同哈希函数对输出序列重新哈希；
  - 调用 PRC 的 Decode 函数判断能否恢复原码字；
  - 若能恢复，则认为检测到水印。

- **理论构造基础**：
  - 基于 **次指数级学习噪声（LPN）假设** 构造二进制 PRC；
  - 或在更强场景下使用 [CGG+25] 中基于 “permuted codes” 的 PRC 实现更高编辑鲁棒性。

### 3. 方案扩展
论文提出三种理论性水印方案：
1. **仅随机替换鲁棒方案**（基于 LPN 假设）；
2. **随机替换 + 删除鲁棒方案**（需“自然 prompt”假设或更强 PRC）；
3. **支持任意编辑鲁棒方案**（基于 permuted code 假设）。

## 六、主要结论与发现

- **核心结论**：  
  即使 LLM 输出极低熵（例如平均每 token 仅有 1 bit 熵），仍可构造不可检测、鲁棒的水印机制。

- **主要成果**：
  - 基于次指数级 LPN 假设的二进制 PRC，可以支持在常数熵条件下对随机替换的鲁棒水印；
  - 若附加启发性假设或更强 PRC，可同时抵抗删除；
  - 在 permuted codes 假设下可扩展为对小比例任意编辑（插入 / 删除 / 替换）鲁棒；
  - 理论上证明**无须高熵率假设**即可实现水印检测。
